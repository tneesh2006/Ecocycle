# ♻️ EcoCycle

Open source expired product recycling platform

## What it does
- Add expired products (food, medicine, battery etc)
- Get instant recycling method suggestion
- See nearest recycling centers
- Track impact — CO₂ saved, waste diverted

## Tech Stack
- Frontend: React + Vite
- Backend: Python Flask
- Database: SQLite

## How to run

### Backend (Jupyter Notebook)
- Open app.ipynb in Jupyter
- Run the cell
- Server starts at http://localhost:5000

### Frontend
cd frontend
npm install
npm run dev
Open http://localhost:5173

## Categories supported
food, medicine, cosmetics, electronics, 
battery, plastic, paper, glass, 
metal, clothing, dairy, oil, chemicals, paint
